# Project Documentation

## Project Structure
- `index.html`: main HTML page
- `styles.css`: main stylesheet
- `script.js`: JavaScript file
- `README.md`: repository presentation
- `CONTRIBUTING.md`: contribution guide
- `CODE_OF_CONDUCT.md`: repository code of conduct
- `BRANCH_PROTECTION.md`: branch protection notes

## Git Workflow
- one issue = one task
- one branch = one issue
- one pull request per branch
- merge into `main` after review

## Templates
- issue templates are stored in `.github/ISSUE_TEMPLATE/`
- pull request template is stored in `.github/pull_request_template.md`

## Branch Protection
- pull request required before merging into `main`
- at least one approval required
